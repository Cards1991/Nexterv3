Atualizar Funcionários via Excel

Selecione um arquivo Excel (.xlsx) para atualizar os dados dos funcionários existentes. O sistema usará a coluna 'cpf' para encontrar os funcionários e atualizar seus dados.

As colunas no arquivo Excel devem ter os nomes exatos conforme a documentação. Apenas os funcionários encontrados pelo CPF serão atualizados.
